Margin, Border and Padding

Margin

in code a marin is "invisible" space at the outside edge of each elements content that helps seperate elements from each other, it occurs after border and padding. You can create more space between elements by increasing the Margin. Some elements have margins by default, others you can give margins through CSS styling

Border

in code the Border is the space between the margin and the Padding

Padding

in code a padding is "invisible" space at the inside edge of each element that seperates elements from each other. You can create more space inside elements by increasing the padding. Some elements have padding by default, others you can give padding through CSS styling
